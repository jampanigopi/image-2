What's the Difference?

HTML
Hyper Text Markup Language
    create the structure
  • controls the layout of the content
  • provides structure for the webpage design
  • the fundamental block of any webpage






CSS
Cascading Style Sheet
    Stylize the website
  • Applies style to the web page elements
  • Target various screen sizes to make web pages responsive
  • Primarly handles the "look and feel"of a web page






JS
Java script
    Increase the activity
  • adds interactivity to a webpage
  • Handles complex functions and features
  • Programmatic code which enhances the functionality
